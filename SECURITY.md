# Security Policy

## Supported versions
We currently support and patch the latest main branch. Releases older than the most recent minor version may not receive security updates.

## Reporting a vulnerability
Please email idevhub@proton.me with a detailed description of the issue and steps to reproduce it. We aim to acknowledge receipt within two business days and provide an initial assessment within five business days.

If the vulnerability is sensitive, consider encrypting your report using our PGP key (coming soon).

## Disclosure process
- We will work with you to understand and resolve the issue quickly.
- A fix will be prepared and released as soon as possible.
- We will credit reporters in the release notes unless you prefer to remain anonymous.
